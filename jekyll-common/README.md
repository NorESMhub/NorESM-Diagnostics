## configuration options

To configure author information, add
```yaml
author:
  name: 'Author'
  link: 'https://url.tld'
```
to your `_config.yml`.

The source code license can be set by
```yaml
license:
  code: 'Apache-2.0'
```
or
```yaml
license:
  code: 'MIT'
```
